üîê Complete Encryption System

Encrypt Current Image
Encrypts image in BOTH locations (tool + source folder). Image becomes unviewable without password.

Decrypt for Analysis
Temporary decryption for analysis only. Original remains encrypted.

Encrypt Entire Folder
Encrypts ALL images in BOTH locations. Cannot view any images without password.

Permanently Decrypt
Restores image in BOTH locations. Image becomes viewable again.

Decrypt All
Permanently decrypts ALL encrypted images in the case.

üîç Integrity Verification (SHA-256)

Hash Check
Verifies individual image integrity. Compares current hash with baseline. Shows: MATCH (original), MODIFIED (tampered), or FIRST (new baseline).

Case Hash
Verifies integrity of ENTIRE case. Detects if any images were added/removed/modified.

Auto-Hash
Automatically saves hash when image is loaded. First image creates baseline.

Noise Analysis
